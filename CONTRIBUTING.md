# Contributing

Thanks for helping improve @agentwatch/parsing. This project uses Bun and TypeScript.

## Development setup

```bash
bun install
bun run typecheck
bun test
bun run build
bun run test:node
bun run test:coverage
```

## Pull requests
- Keep changes focused and explain the intent.
- Add or update tests for behavior changes.
- Update docs or CHANGELOG.md when user-facing behavior changes.

## Issues
- Include clear steps to reproduce.
- Provide minimal sample transcripts if relevant (redact secrets).
